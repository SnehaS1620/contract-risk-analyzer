# AI Contract Risk Analyzer for SMEs

This tool helps small and medium businesses understand legal risks in contracts.

## Features
- Clause extraction
- Risk scoring (Low/Medium/High)
- Plain-language explanations
- Hindi & English support
- PDF and CSV export
- Works even without AI

## Tech Stack
Python, Streamlit, Rule-based NLP, GPT

## Live Demo
https://contract-risk-analyzer-grcsgahpph3crbfl6jumcy.streamlit.app/

pip install -r requirements.txt
streamlit run app.py
